## Setup

### 1. Install dependencies

```sh
$ npm install
```
### 2. Run json-server

```sh
$ json-server --watch src/db.json --port 3001
```

### 3. Run the app

```sh
$ npm start
```

### This project uses the Flux architecture

In a minimal version, the Flux architecture can contain three layers interacting in order:
- Actions
- Stores
- Views
